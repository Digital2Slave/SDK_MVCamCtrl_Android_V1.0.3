byte []  pData,

)
static

获取相机属性树XML

参数
handle[IN] 设备句柄
data[OUT] XML数据接收缓存
dataLen[OUT] 实际数据大小
返回
成功，返回MV_OK；失败，返回错误码 。
备注
 • 当pData为NULL或nDataSize比实际的xml文件小时，不拷贝数据，由pnDataLen返回xml文件大小；
 • 当pData为有效缓存地址，且缓存足够大时，拷贝完整数据保存在该缓存里面，并由pnDataLen返回xml文件实际大小。
 • 接口所在类：MvCameraControl
