工业相机Android SDK开发指南V1.0.0（java）

概述

  • 随着Android系统及嵌入式处理器在工业领域的应用越来越广，市场上对于Android系统对接工业相机的需求越来越多。需要开发Android版本工业相机SDK，提升SDK对不同平台的兼容性。
  • Android工业相机SDK是针对Android系统平台开发SDK，其对外提供的接口为Java接口，支持标准GigE、U3V相机的取流和参数设置。用户可以使用本产品进行Android应用软件的二次开发。
  • 工业相机SDK主要有参数配置和图像采集两个基本功能，主要作用是实现对相机的控制，提供了相机参数设置、图像采集和图像处理等接口。
