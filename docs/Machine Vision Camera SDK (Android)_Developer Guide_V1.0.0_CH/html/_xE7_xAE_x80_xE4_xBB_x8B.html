工业相机Android SDK开发指南V1.0.0（Java）

概述

随着Android系统及嵌入式处理器在工业领域的应用越来越广，市场上对于Android系统对接工业相机的需求越来越多，因此开发Android版本的工业相机SDK，来提升SDK对不同平台的兼容性。Android工业相机SDK是针对Android系统平台开发的SDK，其对外提供的接口为Java接口，支持标准GigE、U3V相机的取流和参数设置。用户可以使用本产品进行Android应用软件的二次开发。工业相机SDK主要有参数配置和图像采集两个基本功能，主要作用是实现对相机的控制，提供了相机参数设置、图像采集和图像处理等接口。

注意事项

  • 本手册中提供的代码段仅作为示例代码提供。不正确的代码可能会导致您的相机功能与预期不同，并可能危及您的应用程序。

运行环境

  • 硬件环境：RK3399,RK3128
  • 软件环境：Android 4.4-Android 9.0

  • 依赖库：

        —GenlCam library，工业相机标准通信协议库。

        —MvCameraControl，是对各种协议相机进行封装（目前支持GigE和Usb相机），屏蔽协议交互细节，向二次开发者提供统一的调用接口，实现主机与相机的连接、控制和取流。

        —MVGigEVisionSDK，GigE模块封装了GigEVision协议的具体细节，针对网口相机提供发现相机，打开相机，开始取流，停止取流等接口。

        —libMvUsb3vTL，U3V模块封装了USB3Vision协议的具体细节，针对U口相机提供发现相机，打开相机，开始取流，停止取流等接口。
